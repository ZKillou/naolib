# Commandes via le PIPE

> Un argument vide doit être représenté par un espace

## ADD

`ADD|num?|dest|rebound|time|reset`

- num?: str
- dest: str
- rebound: bool
- time: int
- reset: bool

## SET

`SET|idx|num?|dest|rebound|time|reset`

- idx: int
- num?: str
- dest: str
- rebound: bool
- time: int
- reset: bool

## CLR

`CLR|`
